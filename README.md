6. triangle
   Triangulasi adalah suatu metode yang menghubungkan sekumpulan triangle secara grafis untuk menggambarkan suatu surfaces dan/atau volume. Data yang di-triangulasi disimpan dalam triangle file dalam direktori ./triangle pada project anda. Tiap triangle file memiliki file yang berakhiran dengan .tfl. Struktur dan karekteristik triangle file hampir sama dengan design file. Seperti juga design file, sebuah triangle file dibuat dari satu atau lebih layer-layer . Ketika anda membuat sebuah triangulasi, anda perlu memasukkan nama output triangle file juga output layer.
